Hydrogen & Carbon Fiber lab plan for CRIT CBN

- Layout
  - 16-36 tiles per floor; 2-3 floors 
  - Each corner along with the center is an elevator
  - Final floor is a massive open room and is the showcase for the largest projects

- Rooms
  - Wind Tunnel (24x24 )
    - Mainly for planes & gliders; Will contain CRIT carbon fiber solar wings for the most part
    - May also contain one experiemental aircraft if by an elevator

  - Hydrogen & Blimp Room ( 24x24 )
    - Room filled with electrolysis machines & Hydrogen tanks
    - Also has a water supply; probably just a tank realistically
    - Planned integration with fluid grid but requires player to restore power to the overmap
    - Contains CRIT solar blimp balloons
    - If near an elevator -> finished micro blimp

  - Oxygen Room ( 24x24 )
    - Due to using electolysis to get hydrogen tons of extra oxygen
    - Contains many oxygen tanks
    - I still need to think of a use for it beyond obscene amounts of oxygen

  - Researcher house ( 24x24 )
    - 3 sections -> living space; office; bedroom
    - All should be dynamic mapgen
    - Offices should contain research paper fragements
      - Multiple fragements required to make the paper -> reward with high power recipe
    - Offices should have 1 or 2 keycards
      - Use place_item to spawn -> will not lower spawn rate
    - Living spaces can include anything; These will be the same across all CRIT labs; planned are
      - TV & Couch ( the boring one )
      - Sparring dummy & melee weapons ( include one of the crit ones )
      - Small shooting range ( include 1 crit exclusive gun & one energy gun )
      - Pet area; Contains a few friendly dogs & cats ( might be later )
      - Computer room; Contains multiple computer books & multiple computersi ( see below for uses )

  - Engine Room ( 24x24 )
    - Contains multiple experiemental hydrogen engines
    - Chance: Contains a weak plutonium engine and a note about XEDRA forcing the researchers to find a use for plutonium
    - Compressed hydrogen tanks -> Used as magazines for ammo along with vehicle fuel tanks
    - Contains one engine on a cart w/ alternator for stress testing

  - Gun Room ( 24x24 )
    - Contains
      - Lazer Weapons
        - Just update and include the CRIT ones already in existance
      - Hydrogen Weapons
        - Smaller ones mainly -> Pistol & rifle; Big held ones are covered by base game 

  - Barracks ( 24x24 )
    - Contains
      - Row of 2x2 rooms; Bed & locker with crit solider clothes
      - 1 or 2 bigger 4x4 rooms; Contains lone wolf / enforcer armor
      - Eating area with MREs in the middle
      - Showers between the 4x4 rooms

  - Research Assistant Quarters ( 24x24 )
    - Contains
      - Row of 3x3 rooms; Bed, Locker & Table this time Crit formal wear
      - 2 bigger 4x4 rooms; Contains Medic & Engineer stuff
      - Eating area with fresh fruit & nuts
      - Showers between the 4x4 rooms

  - Corridoors ( 24x24 )
    - Will always be sided by 7x7 rooms ( see side rooms )
    - Will always contain at least one patrolling military zombie
    - Will also contain multiple ()hacks

  - Side Rooms ( 7x7 )
    - Storage Room
      - Contains
        - Carbon fiber
        - Solar cells
        - Electronic bits
        - Hydrogen tanks
    - 3d printing room
      - Multiple 3d printers
      - Carbon fiber in each printer
      - Each printer also contains one of
        - Carbon fiber vehicle part
        - Carbon fiber weapon
        - Carbon fiber armor piece ( new! )
    - Compressor room
      - Contains a gas compressor ( allows reloading of hydrogen canisters )
      - Contains compressed hydrogen tanks ( seperate from hydrogen canisters )
    - Containment room
      - Made as the apocalypse began sealed steel room
      - Contains one of the nasty zombies and also is one of the other side rooms with double loot
    - Security room
      - Contains 1 or 2 security zombies
      - Contains a security console from which you can see from every security camera
      - Basically get view of all the rooms
    - Power room
      - Contains power substations
      - Contains 1 hydrogen generator ( currently unused in furniture form )
      - Can also turn on/off lights in the area
      - Rare chance to contain a note -> Note will mention cutting power conenctions
    - Server room
      - Contains a few computers with research papers
      - Will also contain a zombie technician
      - Will contain a computer which will note nearby crit labs
    - Autodoc room
      - Contains an autodoc and a lot of medical supplies
      - Can contains some of the CRIT medic exclusive stuff


- Special Things
  - Computers
    - Computers will be computer skill checks; If you do them right they will give you research papers
    - This will be the primary source of research papers; Only printed/half written fragements will be on researchers desks
  - Lazer Turrets
    - Small lazer turrets will appear throughout instead of bullet turrets
    - Basically testing the adaptability of the player and is how it is restricting lower level characters
    - Will reward with plutonium cells & parts for lazer guns
  - Elevators
    - At the center and each of the four corners there will be an elevator
    - You must reach the elevator and put a keycard in on each floor to unlock it
    - The central elevator goes to the surface; The side elevators go to the subway
